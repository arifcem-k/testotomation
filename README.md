# test-automation
This project contains UI test automation scenarios developed using Selenium and TestNG.
---
# Purpose
The main goal of this project is not only to write working tests, but to build a more maintainable and structured test automation approach.
---
# What I Focused On
- Writing more readable and structured test cases
- Improving assertion usage
- Creating meaningful test scenarios instead of basic checks
- Making tests easier to maintain
## Before vs After
This project includes both the initial version and the refactored version of the test code.
- **Before:** Basic test scripts focused on functionality  
- **After:** Structured TestNG-based approach with improved readability and maintainability
